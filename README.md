### Setup .env file

```js
PORT = 5000;
MONGO_URI = your_mongo_uri;
NODE_ENV = development;
JWT_SECRET = your_jwt_secret;
TMDB_API_KEY = your_tmdb_api_key;
```

### Run this app locally

```shell
npm run build
```

### Start the app

```shell
npm run start
```
